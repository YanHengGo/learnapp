# design/ フォルダ 命名ルール

## ファイル名プレフィックス

| プレフィックス | 用途 | 例 |
|---|---|---|
| `design_` | 画面・機能の設計書 | `design_home_navigation.md` |
| `plan_` | 実装計画（タスク単位） | `plan_privacy_policy.md` |
| `guide_` | 技術ガイド・比較資料 | `guide_state_management.md` |
| `api_` | API仕様・リファレンス | `api_reference.md` |
| `tasks_` | タスク管理・バックログ | `tasks_backlog.md` |
