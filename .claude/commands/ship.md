以下の手順を順番に実行してください。

## 1. 変更内容を確認

以下のコマンドを並列で実行して変更内容を把握してください：
- `git status`
- `git diff HEAD`
- `git log --oneline -5`

## 2. コミットメッセージを自動生成

差分を分析して、以下のルールで日本語のコミットメッセージを作成してください：
- 1行目：変更の要約（50文字以内）
- 必要に応じて改行後に詳細を追記
- 末尾に必ず以下を付与：
  ```
  Co-Authored-By: Claude Sonnet 4.6 <noreply@anthropic.com>
  ```

## 3. ステージングしてコミット

```bash
git add -A
git commit -m "<生成したコミットメッセージ>"
```

## 4. プッシュ

現在のブランチにプッシュしてください：
```bash
git push origin <現在のブランチ名>
```

## 5. PRコメント投稿

まず現在のブランチにオープンなPRが存在するか確認してください：
```bash
gh pr view --json number,title 2>/dev/null
```

**PRが存在する場合：**
レビュワー向けに以下の形式で日本語コメントを投稿してください：
```bash
gh pr comment <PR番号> --body "<コメント>"
```

コメントの内容：
- 変更の概要
- 主な変更ファイル一覧
- レビュー時の注意点（あれば）

**PRが存在しない場合：**
PRを作成してください：
```bash
gh pr create --title "<コミットメッセージの1行目>" --body "<変更概要を日本語で記載>"
```

各ステップの完了をユーザーに報告しながら進めてください。
